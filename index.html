HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Element, refers to the entire element from opening tag to closing tag
  • Tag, refers only to what appears inside the angle brackets

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • text-align is one way to center a title or heading on a webpage.
  • img selector assigns property values to any image element in the markup.
  • display property is assigned with the block value, which assigns each image to start on a new line.

Git

  • git status: check what branch we are currently on
  • git checkout - b branch-name: creates a new branch and switches to it
  • git add [file]: add a file as it looks now to your next commit
  • git commit -m "[descriptive message]": commit your staged content as a new commit snapshot

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • To connect script.js to index.html, we have to add it under the footer element and above the closing element for body.
  • console.log method was created to make it easier for developers to test their code.
